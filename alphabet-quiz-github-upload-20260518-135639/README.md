# Alphabet-Quiz

Eine einfache React- und TypeScript-Web-App für erwachsene Spracheinsteiger in der Alphabetisierung.

Die Lernenden üben die Reihenfolge der Buchstaben im deutschen Alphabet ohne Umlaute.

## Funktionen

- 4 Level
- 10 Runden pro Level
- große Schrift und große Eingabefelder
- einfache deutsche Sprache
- direkte Rückmeldung bei jeder Antwort
- Auswertung nach Level 4
- Pokal und Konfetti bei mehr als 80 Prozent richtig

## Lokal starten

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
